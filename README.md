RPCLS
=====

Install Cordova, make sure you have [Node.js](http://nodejs.org/) installed, then run
```CLI
$ sudo npm install -g cordova
```
Install Ionic
```CLI
$ sudo npm install -g ionic
```

Usefull commands

Configure Platforms
```CLI
$ ionic platform ios
$ ionic platform android
```

Test it out
```CLI
$ ionic build ios
$ ionic emulate ios
```
